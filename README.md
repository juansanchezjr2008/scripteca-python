<h1>Mi Scripteca Python</h1>
<p>
Este repositorio es mi biblioteca personal de scripts en Python, creados para resolver problemas reales, automatizar tareas y experimentar con ideas creativas.  
Cada script incluye:
</p>
<ul>
  <li>Explicación clara de su lógica y propósito</li>
  <li>Ejemplos prácticos de uso</li>
  <li>Versiones en <strong>Jupyter Notebook</strong> para ejecutar, modificar y aprender fácilmente</li>
</ul>
<p>
La idea es que cualquiera pueda entender y adaptar el código, ya seas principiante aprendiendo conceptos o alguien que solo necesita soluciones rápidas para sus proyectos.
</p>
